# 音频加密解密工具 v2.0

一个功能强大的HTML5音频加密解密工具，支持多种加密格式，提供自动后台解密播放功能。

## 🌟 功能特性

### 基础功能
- **设备码管理**：支持生成随机设备码或输入自定义设备码
- **密钥生成**：基于设备码生成安全的加密密钥
- **音频加密**：支持加密音频文件为多种加密格式
- **音频解密**：支持解密多种加密格式的音频文件
- **文件管理**：支持处理后文件的下载和播放
- **进度显示**：实时显示加密/解密进度

### v2.0 新增功能
- **自动后台解密**：支持自动检测加密音频并在后台解密播放
- **无缝播放体验**：无需手动操作，直接播放加密音频
- **文件拖放支持**：可将加密音频直接拖放到页面上播放
- **解密队列管理**：支持多文件同时处理，自动排队
- **实时状态显示**：显示解密进度和状态信息

## 📋 支持格式

### 输入格式
- 普通音频：MP3、FLAC、WAV
- 加密音频：SMP3、SFLAC、LGG、LAV

### 输出格式
- 加密格式：SMP3（MP3加密后）、SFLAC（FLAC加密后）、LGG、LAV
- 解密格式：MP3、FLAC、WAV（根据原格式自动判断）

## 📁 文件大小限制

- 最大支持250MB的音频文件

## 🚀 使用方法

### 1. 密钥生成

1. 打开 `audio_encryptor.html` 文件
2. 在「设备码与密钥管理」部分，点击「生成设备码」或输入自定义设备码
3. 点击「生成密钥」按钮，系统将生成加密解密所需的密钥
4. 密钥将显示在密钥显示区域

### 2. 音频加密

1. 在「音频文件处理」部分，点击「选择音频文件」按钮
2. 选择要加密的音频文件（MP3、FLAC、WAV格式）
3. 点击「加密」按钮，系统将开始加密
4. 加密完成后，处理后的文件将显示在「处理后的文件」列表中
5. 可以点击「下载」按钮保存加密文件，或点击「播放」按钮直接播放

### 3. 音频解密

1. 在「音频文件处理」部分，点击「选择音频文件」按钮
2. 选择要解密的加密音频文件（SMP3、SFLAC、LGG、LAV格式）
3. 点击「解密」按钮，系统将开始解密
4. 解密完成后，处理后的文件将显示在「处理后的文件」列表中
5. 可以点击「下载」按钮保存解密文件，或点击「播放」按钮直接播放

### 4. 自动后台解密播放（v2.0新功能）

1. 确保已生成密钥
2. 在「音频播放」部分，点击「开启自动解密」按钮
3. 直接将加密音频文件拖放到页面上，或在音频播放器中加载加密文件
4. 系统将自动检测加密音频，在后台解密并播放
5. 解密状态将显示在「解密状态」区域

## 💻 技术实现

- **前端框架**：纯HTML5 + CSS3 + JavaScript
- **加密算法**：AES-GCM 256位加密
- **密钥生成**：基于SHA-256哈希算法
- **文件处理**：使用HTML5 File API和Web Crypto API
- **音频播放**：HTML5 Audio API

- Edge 79+

## 📝 注意事项

1. 密钥是基于设备码生成的，同一设备码生成的密钥相同
2. 请妥善保管设备码和密钥，丢失将无法解密文件
3. 加密后的文件无法直接播放，需要使用本工具解密或开启自动解密功能
4. 自动解密功能需要先生成密钥才能使用
5. 建议在使用前备份原始音频文件
