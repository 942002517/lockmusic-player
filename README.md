# 音频加密解密工具

保护您的音频文件安全，提供无缝播放体验

## 📖 产品简介

音频加密解密工具是一款基于HTML5技术开发的Web应用，用于加密和解密音频文件，支持多种加密格式，提供自动后台解密播放功能。

该工具采用先进的AES-GCM 256位加密算法，确保您的音频文件安全可靠。同时，提供了现代化的用户界面和流畅的用户体验，支持基础模式和高级模式切换，满足不同用户的需求。

## 🚀 使用方法

### 基础使用流程
1. 打开音频加密解密工具（`audio_encryptor.html`）
2. 系统会自动生成设备码和5个密钥（首次启动时）
3. 在「音频文件处理」部分选择要处理的音频文件
4. 点击「加密」或「解密」按钮进行处理
5. 处理完成后，可以在「处理后的文件」列表中下载或播放文件

### 自动后台解密播放
1. 确保已生成密钥（首次启动时自动生成）
2. 在音频播放器中点击「开启自动解密」按钮
3. 直接加载加密音频文件，系统会自动在后台解密并播放

### 高级模式使用
1. 在音频播放器部分点击「高级模式」按钮切换到高级播放器
2. 可以查看歌曲元数据、调节进度和音量
3. 点击右上角的⚙️图标打开设置面板
4. 在设置面板中可以管理设备码和密钥

### 歌曲列表使用
1. **自动扫描导入**：
   - 页面加载时自动从本地存储加载已保存的歌曲列表
   - 无需手动操作，自动完成

2. **手动扫描导入**：
   - 点击「手动扫描导入」按钮
   - 选择一个或多个音频文件（支持非加密和加密格式）
   - 歌曲将被添加到歌曲列表中

3. **文件夹上传导入**：
   - 点击「上传文件夹导入」按钮
   - 选择包含音频文件的文件夹
   - 文件夹中的所有音频文件将被自动添加到歌曲列表中
   - 自动过滤非音频文件，只导入支持的音频格式
   - 跳过重复文件，避免列表冗余
   - 支持嵌套文件夹结构（部分浏览器可能不支持）

4. **歌曲列表管理**：
   - 在「歌曲列表」区域查看已导入的歌曲
   - 点击「播放」按钮直接播放选中歌曲
   - 点击「下一首」按钮将歌曲设为下一首播放
   - 点击「删除」按钮移除不需要的歌曲
   - 查看歌曲名称、大小和加密状态
   - 点击「播放全部」按钮播放列表中的所有歌曲
   - 点击「清空列表」按钮清空所有歌曲

5. **播放控制**：
   - 播放结束后自动播放下一首歌曲
   - 在「当前播放」区域查看实时播放信息
   - 点击「停止」按钮停止当前播放

6. **多选文件处理**：
   - 在文件选择对话框中按住Ctrl或Shift键选择多个文件
   - 支持同时处理多个音频文件
   - 文件信息区域显示选中文件的总数和总大小

## ⚠️ 注意事项

### 重要提示
- 密钥是基于设备码生成的，同一设备码生成的密钥相同
- 请妥善保管设备码和密钥，丢失将无法解密文件
- 加密后的文件无法直接播放，需要使用本工具解密或开启自动解密功能
- 自动解密功能需要先生成密钥才能使用
- 建议在使用前备份原始音频文件
