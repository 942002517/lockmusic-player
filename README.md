# LockMusic Player - 音频加密解密工具 v3.1

LockMusic Player 是一个功能强大的音频加密解密工具，支持多种音频格式的加密和解密，内置音频播放器，以及集成了 Unlock Music 工具用于解密各种平台格式的音频文件。

## 功能特点

### 1. 音频文件处理
- 支持加密音频文件（最大250MB）
- 支持解密音频文件
- 支持一键解密功能
- 支持文件夹上传和批量处理
- 自动匹配密钥解密
- 支持多种音频格式：MP3, FLAC, WAV, SMP3, SFLAC, LGG, LAV

### 2. 密钥管理
- 生成和管理设备码
- 生成和保存密钥
- 支持自主添加密钥（最多50个）
- 自动匹配密钥解密
- 一键保存当前密钥
- 重新生成密钥功能

### 3. 高级音频播放器
- 支持播放多种音频格式
- 播放进度显示和控制
- 音量滑动调节和显示
- 播放/暂停、快进/快退控制
- 实时显示播放状态
- 自动解密加密音频

### 4. 平台格式解密
- 集成 Unlock Music 工具
- 支持多种平台加密格式：
  - QQ音乐：QMC3, QMC0, QMCFLAC, QMCOGG, QMCWAV, QMCWM, MFLAC, MGG, MFLAC0, MGG1, MGGL, MGALAXY, MFLACH
  - 网易云音乐：NCM
  - 酷狗音乐：KGM, VPR, KGG
  - 酷我音乐：KWM
  - 虾米音乐：XM
  - 喜马拉雅：X2M, X3M
  - 咪咕音乐：MG3D
  - 蜻蜓FM：QTA

### 5. 处理后的文件管理
- 显示处理后的文件列表
- 支持单个文件下载
- 支持批量下载所有文件
- 支持清除处理记录
- 播放处理后的文件

## 使用说明

### 1. 设备码和密钥管理
- 生成设备码：点击"生成设备码"按钮
- 生成密钥：输入设备码后点击"生成密钥"按钮
- 保存密钥：密钥自动保存到本地，下次无需重新生成
- 添加密钥：在设置面板中输入32位十六进制密钥，点击"添加密钥"
- 管理密钥：在设置面板中可以复制、删除密钥，或重新生成密钥

### 2. 音频文件处理
- **选择文件**：点击"选择音频文件"或"选择文件夹"
- **加密文件**：选择文件后点击"加密"按钮
- **解密文件**：选择加密文件后点击"解密"按钮
- **一键解密**：选择文件或文件夹后点击"一键解密"按钮
- **自动解密**：播放加密文件时自动尝试解密

### 3. 音频播放
- **加载音频**：点击播放器中的"📁"按钮选择音频文件
- **播放控制**：使用播放/暂停、快进/快退按钮控制播放
- **进度控制**：点击进度条调整播放进度
- **音量控制**：拖动音量滑块调整音量，实时显示音量百分比
- **状态显示**：实时显示播放、加密、解密状态

### 4. 平台格式解密
- 直接使用内置的 Unlock Music 工具解密音乐文件
- 支持多种平台格式，无需额外安装软件

### 5. 处理后的文件管理
- **查看文件**：处理后的文件会显示在"处理后的文件"列表中
- **下载文件**：点击文件右侧的"下载"按钮下载单个文件
- **批量下载**：点击"批量下载"按钮下载所有处理后的文件
- **清除记录**：点击"清除记录"按钮清空处理后的文件列表
- **播放文件**：点击文件右侧的"播放"按钮播放文件

## 技术实现

- **前端框架**：原生 HTML, CSS, JavaScript
- **加密算法**：AES-GCM
- **并发处理**：支持最大3个并发解密任务
- **本地存储**：使用 localStorage 保存密钥和设置
- **音频播放**：使用 HTML5 Audio API
- **文件处理**：使用 File API 和 Blob API
- **集成工具**：嵌入 Unlock Music iframe
